Upgrade Routine v1.0 => v1.1 ONLY:

Make sure you have a structural and schematic backup before you begin.

Please read these instructions carefully. Upgrading is entirely at your own risk:

Upgrade Instructions:

To upgrade to v1.1, follow the instructions below. Structure has been maintained and where a folder only is shown, you should replace the whole folder unless stated otherwise.

1: Replace or add the following files/folders:

/admin/ (Updated)
/classes/fonts/ (New)
/classes/class_mail.inc.php (Updated)
/classes/php-captcha.inc.php (New)
/templates/popups/contact.tpl.php (Updated)
/templates/popups/friend.tpl.php (Updated)
index.php (Updated)

If you had changed some of the user editable vars in the index.php file, you may have to re-edit them.

2: Go to your control panel and access your recipe database. Run the following file to update your tables:

/docs/upgrade_data/v1_1/upgrade.sql

NOTE: If your prefix is different from the default 'mr_', you must update this file BEFORE you run it.

3: Open your language file and add the new variables that are shown in the following file:

/docs/upgrade_data/v1_1/new_lang.txt

Copy and paste above the closing ?> tag. Or simply overwrite your current language file with the new one.

Thats it, all done. Check your system to make sure everything is working ok. If you wish to enable the SMTP mail options log into your admin area and update your settings. The captcha can be disabled as per the F.A.Q instructions.
